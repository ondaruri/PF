# みんなのラーメン

## サイト概要
ラーメン情報投稿サイト

### サイトテーマ
美味しいラーメンを紹介し合おう！
誰でも自由にラーメンの写真を投稿、レビューを行い、ラーメン業界を盛り上げる。

### テーマを選んだ理由
星の数ほどあるラーメン屋。隠れた名店を見つけるために作成。


### ターゲットユーザ
ラーメンマニア、一人暮らし男性、ラーメン屋関係者、一般ユーザ

### 主な利用シーン
・外食時のお店検討に
